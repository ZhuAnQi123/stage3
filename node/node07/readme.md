##### websocket 实现长连接 
任何一个长连接 由俩部分组成 客户端 服务端

服务端
1. 引入模块 创建长连接服务器
2. 监听客户端连接 
3. 向客户端发送消息
4. 接收消息
5. 服务主动断开连接
6. 监听客户端断开连接
客户端
1. 创建长连接对象
2. 与服务端进行连接
3. 接受消息
4. 向服务端发送消息
5. 处理服务器端断开连接
6. 主动断开连接
